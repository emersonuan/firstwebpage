🚀 Tutorial Paso a Paso

1. Preparación inicial

• Crea cuenta en GitHub
• Instala Git
• Elige editor de código (VS Code recomendado)

2. Generación de código con IA

Ejemplo de prompt para ChatGPT:
"Genera código HTML/CSS para una landing page moderna con menú lateral desplegable, secciones para contenido de IA y estilos responsivos"

3. Configuración de GitHub

# Inicializar repositorio git init # Conectar repositorio remoto git remote add origin https://github.com/tuusuario/turepositorio.git # Primer commit git add . git commit -m "Initial commit" git branch -M main git push -u origin main

4. Publicación con GitHub Pages

  1. Ve a la configuración de tu repositorio
  2. Selecciona la pestaña "Pages"
  3. En "Branch" selecciona "main" y /root
  4. ¡Tu sitio estará disponible en:
    https://[usuario].github.io/[repositorio]
